# Тренировки по алгоритмам 3.0

Здесь выложены мои решения части задач яндексовских тренировок 3.0, 
прошедших в феврале-марте 2023 г. Все задачи решались в формате контеста без
ограничений по времени, кроме финального спринта, на который отводилось 3 часа.

Я буду постепенно выкладывать условия к задачам в виде readme

### Примечание к решениям

Формат контеста не предполагает других критериев оценки решений, кроме работоспособности
и эффектисвости алгоритма, поэтому в коде нет комментариев, и не всегда хороший нейминг.
Также в большинстве случаев опущена проверка правильности ввода, поскольку формат вводных
данных на тестах гарантирован по условию.
